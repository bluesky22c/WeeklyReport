  • HIOKI IM3533 실응답·개방/단락/실저항 정상 확인, USB 허브/FTDI 변환기 이슈 진단

3. 코드 정리

  • 옛 스코프 시뮬레이션 등 미사용 코드 7개 제거 (커밋 대기)

4. 문서·구조 정리

  • 통신 구조 = 측정PC ↔ SFA 제어PC CC-Link IE(각 PC Q80BD) 정리
  • 계측기 테스트 정리 문서(MD+HTML), 측정 PC 셋업 안내 폴더 작성

5. Set up 프로그램 설치

  • 측정 PC에 프로그램 설치: ① .NET 8 Desktop Runtime → ② DCAS exe 복사
